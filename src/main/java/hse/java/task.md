Выражение может быть операцией или константой 

Строковое представление константы - значение константы, вычисление константы - значение константы

Мы рассматриваем операции с двумя операндами или бинарные операции

Бинарная операция - это результат применения операции к двум операндам

Строковое представление бинарной операции - (строковое представление левого операнда символ операции строковое представление правого операнда)
Пример: ((1+2)*(2+1))

Вычисление - функция, которая должна возвращать результат выражения

Виды операций:

- Sum - сумма двух выражений

- Sub - развность двух выражений

- Mul - произведение двух выражений

- Div - частное двух выражений

Задача заключается в проектировании системы классов, для реализации 4 перечисленных выше операций. Каждая операция должна удовлетворять интерфейсу 
IntExpression. Операции должны называться как в задании (Sum, Sub, Mul, Div). Задачу решаем без Generic, только для целых чисел.
